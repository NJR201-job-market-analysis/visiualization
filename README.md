# 視覺化專案集

這個專案集包含多個視覺化相關的子專案，用於展示職缺技能樹與資料視覺化。

## 專案結構

### 104人力銀行職業探索
位於 `/104人力銀行_職業探索_JobCat` 目錄下，這個專案展示了職業技能的視覺化分析：

1. 技能心智圖（Mind Map）
   - 使用階層式結構展示職業技能關係
   - 互動式視覺化展示
   - 技能分類與關聯分析

2. 技能網絡圖（Network Graph）
   - 展示技能之間的關聯性
   - 節點表示不同技能
   - 連線表示技能間的關係

### 互動式樹狀結構
位於 `/互動式樹狀結構` 目錄下，這個專案展示了如何使用不同的技術來實現互動式的樹狀結構視覺化：

1. 軟體開發技能樹（Topic Tree）
   - 完整的軟體開發技能分類
   - 涵蓋各種程式語言、框架和工具
   - 系統化的技能層級結構

2. 使用 Plotly 實現的樹狀圖（Treemap）
   - 使用 Plotly Express 繪製互動式樹狀圖
   - 資料轉換為 DataFrame 格式進行視覺化
   - 提供豐富的互動功能，如縮放、懸停顯示等

## 技術棧
- Python
- Plotly Express
- Pandas
- NetworkX
- Graphviz
- HTML/JavaScript
- Mermaid.js

## 運行專案
進入相應的專案目錄，使用 Jupyter Notebook 打開並運行相關檔案。
