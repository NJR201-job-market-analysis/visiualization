# 視覺化專案集

這個專案集包含多個視覺化相關的子專案，用於展示職缺類型的資料視覺化。

## 專案結構

### 互動式樹狀結構
位於 `/互動式樹狀結構` 目錄下，這個專案展示了如何使用不同的技術來實現互動式的樹狀結構視覺化：

1. 使用 Tkinter 實現的樹狀結構瀏覽器
   - 基於 Python Tkinter 和 ttk 庫
   - 提供可展開/收合的樹狀節點
   - 適合顯示層級結構資料

2. 使用 Plotly 實現的樹狀圖（Treemap）
   - 使用 Plotly Express 繪製互動式樹狀圖
   - 資料轉換為 DataFrame 格式進行視覺化
   - 提供豐富的互動功能，如縮放、懸停顯示等

## 技術棧
- Python
- Tkinter/ttk
- Plotly Express
- Pandas
